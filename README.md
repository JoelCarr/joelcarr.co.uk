# joelcarr.co.uk

My little space on the web to talk about my work.

## Setup

This site is built using [11ty.io](https://www.11ty.io/) which requires [node and npm](https://nodejs.org/en/).

Install dependencies:

```bash
npm install
```

Run site locally:

```bash
npm run serve
```

Build site for deployment:

```bash
npm run build
```
